---
title: Map Set
---

- **Map -** коллекция ключ/значение, как и Object. Но основное отличие в том, что Map позволяет использовать ключи любого типа. Для работы с Map доступны методы для добавления, удаления, получения элементов, а также определения размера Map и проверки наличия элемента по ключу (set, delete, get, size, has).
- **Set -** особый вид коллекции: "множество" значений (без ключей), своего рода массив, где каждое значение может появляться только один раз.
