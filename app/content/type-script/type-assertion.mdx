---
title: Type Assertion
---

В TypeScript есть возможность выполнить преобразование типов, которое называется **Type Assertion (утверждение типа)** или **Type Casting (приведение типа)**. Type Assertion позволяет явно указать компилятору, что определенное значение должно быть рассмотрено как определенный тип, несмотря на то, что компилятор может иметь другое представление о типе этого значения.

#### Type Assertion можно выполнить двумя способами:

- с помощью оператора **&lt;&gt;** (угловые скобки):
```typescript
let someValue: any = 'hello';
let stringLength: number = (<string>someValue).length;
```
В этом примере переменная someValue имеет тип any, что означает, что она может содержать значения любого типа. Но с помощью оператора **&lt;&gt;** (угловые скобки) и явного указания типа string, someValue должна быть рассмотрена как строка. Затем мы присваиваем длину строки переменной stringLength.

- с помощью оператора **as**:
```typescript
let someValue: any = 'hello';
let stringLength: number = (someValue as string).length;
```
В этом примере используется оператор **as** для выполнения утверждения типа. Оператор **as** следует за значением someValue и указывает, что это значение должно быть рассмотрено как тип string.

Type Assertion полезен в случаях, когда у разработчика есть более точная информация о типе значения, чем может предоставить компилятор. Он позволяет сделать явное преобразование типа и использовать функции и свойства, которые доступны только для указанного типа.
