Массивы в TypeScript определяются с помощью выражения **[]** и также являются строго типизированными. Это означает, что если изначально массив содержит строки, то в дальнейшем он сможет работать только со строками.

```typescript
let list: number[] = [10, 20, 30];
let colors: string[] = ['red', 'green', 'blue'];
console.log(list[0]); // 10
console.log(colors[1]); // 'green'
```

Как и в JavaScript, с помощью индексов можно обращаться к элементам массива. В приведенном примере массив list содержит числа, массив colors содержит строки.

Альтернативный способ определения массивов представляет применение типа **Array&lt;&gt;**, где в угловых скобках указывается тип элементов массива:

```typescript
let names: Array<string> = ['Tom', 'Bob', 'Alice'];
console.log(names[1]); // 'Bob'
```

Фактически такие формы массивов, как **`number[]`** или **`string[]`**, являются сокращением типов **`Array<number>`** или **`Array<string>`**.

#### ReadonlyArray

Массивы в TypeScript по умолчанию могут изменять значения своих элементов. Но TypeScript предоставляет возможность определить массивы, элементы которых нельзя изменять. Для этого применяется тип **ReadonlyArray&lt;&gt;**, в котором в угловых скобках указывается тип элементов массива.

```typescript
const people: ReadonlyArray<string> = ['Tom', 'Bob', 'Sam'];
people[1] = 'Kate'; // Ошибка: элементы массива ReadonlyArray нельзя изменять
```

Массив **ReadonlyArray** поддерживает большинство операций только для чтения, но не позволяет изменять значения элементов, добавлять новые элементы или удалять существующие.

Массивы в TypeScript также поддерживают **декомпозицию** на константы и переменные. Можно использовать оператор **...** для указания массива, в который будут помещены все оставшиеся элементы массива, которые не вошли в предыдущие переменные/константы.

```typescript
const people: string[] = ['Tom', 'Bob', 'Sam'];
const [first, second, third] = people;
console.log(first); // 'Tom'
console.log(second); // 'Bob'
console.log(third); // 'Sam'
```

Массив people разбивается на три константы - first, second и third. Значения массива передаются константам/переменным по позиции, где первая константа получает первый элемент массива, вторая константа - второй элемент и так далее.
