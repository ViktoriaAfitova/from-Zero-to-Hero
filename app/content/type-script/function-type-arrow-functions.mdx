#### Тип функции:

- Каждая функция в TypeScript имеет свой тип, который определяется по комбинации типов параметров и типа возвращаемого значения.
- Тип функции записывается в следующем формате: (параметр1: тип1, параметр2: тип2, ...): тип_результата.
- Например, функция hello, которая не принимает параметры и ничего не возвращает, имеет тип () => void, так как возвращаемое значение void.
- Тип функции может быть использован для определения переменных, констант и параметров.
- Например, переменная message имеет тип () => void и может содержать любую функцию, которая не принимает параметры и не возвращает значение.

```typescript
function hello(): void {
  console.log('Hello TypeScript');
}

let message: () => void = hello;
message();
```

#### Стрелочные функции:

- Стрелочные функции или arrow functions представляют сокращенный синтаксис для определения функций.
- Стрелочные функции записываются в формате (параметры) => тело_функции.
- Например, функция sum может быть определена как (x: number, y: number) => x + y.
- Если стрелочная функция не требует указания типов параметров, то их можно опустить.
- Например, функция square может быть определена как x => x \* x.
- Если функция не принимает параметры, то используются пустые круглые скобки.
- Если функция принимает только один параметр, то скобки можно опустить.
- Многострочные функции могут быть заключены в фигурные скобки.

```typescript
let sum = (x: number, y: number) => x + y;
let square = (x) => x * x;

console.log(sum(15, 35)); // 50
console.log(square(5)); // 25
```

#### Функции как параметры других функций:

- В TypeScript функции могут быть переданы в качестве параметров других функций.
- Для этого используется тип функции.
- Например, функция mathOp принимает три параметра: x, y и operation, где operation имеет тип функции (a: number, b: number) => number.
- При вызове mathOp можно передать функцию в качестве параметра, которая соответствует указанному типу.

```typescript
function sum(x: number, y: number): number {
  return x + y;
}

function mathOp(
  x: number,
  y: number,
  operation: (a: number, b: number) => number,
): number {
  return operation(x, y);
}

console.log(mathOp(10, 20, sum)); // 30
```
