---
title: type aliases
---

Для создания псевдонима типа используется ключевое слово **type**.

#### 1. Псевдоним типа для простого типа данных:

```typescript
type Age = number;
let age: Age = 25;
```

В данном примере создается псевдоним типа Age, который эквивалентен типу number. Затем переменной age присваивается значение типа Age, что в данном случае является числом.

#### 2. Псевдоним типа для сложного типа данных:

```typescript
type Person = {
  name: string;
  age: number;
};

let person: Person = {
  name: 'Tom',
  age: 30,
};
```

В этом примере создается псевдоним типа Person, который представляет объект с двумя свойствами: name типа string и age типа number. Затем переменной person присваивается значение типа Person.

#### 3. Псевдоним типа для комбинации типов данных:

```typescript
type ID = number | string;
let userId: ID = 12345;
let postId: ID = 'abcde';
```

В этом примере создается псевдоним типа ID, который представляет тип данных, который может быть либо числом, либо строкой. Затем переменным userId и postId присваиваются значения типа ID, которые могут быть как числами, так и строками.

#### Расширение псевдонимов

Одни псевдонимы могут заимствовать или расширять код других. Для этого применяется операция **&**. Например:

```typescript
type Person = { name: string; age: number };
type Employee = Person & { company: string };
```

В данном случае псевдоним Employee расширяет псевдоним Person, добавляя к нему свойство company, которое представляет тип string. То есть фактически мы имеем дело с типом:

```typescript
type Employee = { name: string; age: number; company: string };
```
