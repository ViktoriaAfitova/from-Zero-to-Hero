---
title: Variables & constants
---

В TypeScript, переменные и константы объявляются с использованием ключевых слов **let** и **const** соответственно.

#### Переменные (let):

```typescript
let x: number = 5; // объявление переменной x типа number со значением 5
let message: string = 'Hello'; // объявление переменной message типа string со значением 'Hello'
```

#### Константы (const):

```typescript
const PI: number = 3.14; // объявление константы PI типа number со значением 3.14
const MAX_SIZE: number = 100; // объявление константы MAX_SIZE типа number со значением 100
```

_Правильное использование типов и ключевых слов **let** и **const** в TypeScript помогает обеспечить типобезопасность и улучшает понимание кода._
