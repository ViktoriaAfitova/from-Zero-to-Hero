---
title: lifecycle
---

В React компоненты имеют свои жизненные циклы, которые представляют собой последовательность событий, происходящих в течение жизни компонента, начиная с его создания и до удаления из DOM. Жизненные циклы позволяют выполнять определенные действия на разных этапах жизни компонента.

#### Жизненные циклы классовых компонентов

##### Инициализация (Initialization):

- constructor: Вызывается при создании компонента и инициализации его состояния.

##### Монтирование (Mounting):

- render: обязательный метод в классовых компонентах, служит для создания React элементов, представляющих компонент. Он должен быть чистой функцией без побочных эффектов.
- componentDidMount:  уведомляет о том, что компонент соединен с DOM
деревом.

##### Обновление (Updating):

- render
- componentWillReceiveProps: уведомляет щ том, что приходят новые
входящие свойства в компонент.
- shouldComponentUpdate: позволяет оптимизировать производительность компонента, решая, нужно ли повторно рендерить компонент при изменении пропсов или состояния.
- componentWillUpdate: уведомляет, что компонент будет обновлен
- render: повторный рендеринг компонента.
- componentDidUpdate: уведомляет, что компонент был обновлен.

##### Размонтирование (Unmounting):

- componentWillUnmount:  используется для удаления слушателей и очистки
компонента. Вызывается перед удалением компонента.
