---
title: JSX
---

**JSX (JavaScript XML)** - это расширение синтаксиса JavaScript, которое позволяет использовать синтаксис HTML внутри JavaScript.

Браузеры не могут напрямую понимать JSX, поэтому перед выполнением кода в браузере, JSX-код должен быть преобразован в обычный JavaScript. Этот процесс называется транспиляцией.

Реакт-компилятор (например, Babel) выполняет транспиляцию JSX в JavaScript, который браузер может понять. Во время транспиляции, JSX-элементы преобразуются в вызовы функций React.createElement().

Таким образом, транспиляция JSX в JavaScript позволяет использовать удобный синтаксис JSX при разработке на React и обеспечивает браузерную совместимость, преобразуя JSX-код в понятный для браузера JavaScript-код.
