---
title: Components
---

Компонент представляет собой независимую и переиспользуемую единицу кода, которая описывает внешний вид и поведение определенной части пользовательского интерфейса.

Компоненты в React используются для разделения пользовательского интерфейса на отдельные и многократно используемые блоки, что делает код более организованным, легким для понимания и поддержки.

Компоненты могут быть **классовыми** или **функциональными**.

- в классовых компонентах используется класс, который наследуется от базового класса React.Component и содержит метод render(), который возвращает JSX (расширение JavaScript, позволяющее описывать структуру пользовательского интерфейса).

- в функциональных компонентах используется функция, которая принимает входные параметры (props) и возвращает JSX.

Пример классового компонента в React:

```jsx
import React from 'react';

class Welcome extends React.Component {
  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}

export default Welcome;
```

Пример функционального компонента в React:

```jsx
import React from 'react';

function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

export default Welcome;
```

Компоненты могут быть вложенными друг в друга, образуя иерархию компонентов. Они могут принимать входные параметры (props), которые позволяют передавать данные и настройки между компонентами. Компоненты также могут иметь внутреннее состояние, которое позволяет им отслеживать и обрабатывать изменения данных или пользовательских действий.

В React компоненты играют центральную роль и являются основным строительным блоком для разработки интерфейсов. Они позволяют разделить сложные пользовательские интерфейсы на более простые и легко управляемые части, что делает разработку приложений более эффективной и масштабируемой.
