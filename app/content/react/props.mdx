---
title: props
---

**props** — это входные данные React-компонентов, передаваемые от родительского компонента дочернему компоненту. Предназначены только для чтения. Их не следует изменять:

```jsx
// Неправильно!
props.number = 42;
```

Если нужно поменять значение в ответ на пользовательский ввод или ответ сервера, используется state (состояние).
