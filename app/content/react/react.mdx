---
title: React
---

**React** — это JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов. Использует компонентный подход. React разрабатывается и поддерживается Facebook и сообществом отдельных разработчиков. React может использоваться для разработки одностраничных и мобильных приложений.
