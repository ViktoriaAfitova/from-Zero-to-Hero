---
title: Elements
---

**Элементы React** - это объекты, которые описывают, что должно быть отображено на экране. Они представляют структуру и свойства (props) компонентов. Элементы являются неизменяемыми и не содержат состояния или методов жизненного цикла. Они создаются с использованием синтаксиса JSX или с помощью функций React.createElement().

Пример элемента, представляющего кнопку:

```jsx
const buttonElement = <button>Click me</button>;
```

**Компоненты React** - это функции или классы, которые принимают входные данные (props) и возвращают элементы React для отображения на экране. Компоненты являются изменяемыми и могут иметь состояние, методы жизненного цикла и другую логику. Компоненты могут быть функциональными (функции) или классовыми (классы, наследующиеся от React.Component). Пример функционального компонента:

```jsx
import React from 'react';

const ButtonComponent = () => {
  return <button>Click me</button>;
}
```

В этом примере ButtonComponent - функциональный компонент, который возвращает элемент кнопки. Компоненты могут использовать элементы и другие компоненты для создания иерархии пользовательского интерфейса.

Элементы React представляют структуру и свойства компонентов, в то время как компоненты React представляют поведение, состояние и методы обработки событий. Компоненты создаются для управления элементами и обновления пользовательского интерфейса в зависимости от внутреннего состояния и свойств (props).
