---
title: Arrow function vs Function
---

1. синтаксис
1. лексическое значение **this** (у стрелочных функций нет своего **this**, если идёт обращение к нему, то берётся снаружи)
1. не имеют **arguments**
1. не могут быть вызваны с оператором **new** в качестве конструктора
