---
title: Logical operators
---

1. Логическое **И (&&):** Возвращает true, если оба операнда являются истинными, и false в противном случае.

```javascript
const a = true;
const b = false;
console.log(a && b); // Output: false
```

2. Логическое **ИЛИ (||):** Возвращает true, если хотя бы один из операндов является истинным, и false в противном случае.

```javascript
const a = true;
const b = false;
console.log(a || b); // Output: true
```

3. Логическое **НЕ (!):** Инвертирует значение операнда. Если операнд был true, то после применения логического НЕ он становится false, и наоборот.

```javascript
const a = true;
console.log(!a); // Output: false
```

4. **Оператор нулевого объединения (??):** Возвращает значение правого операнда, если левый операнд имеет значение null или undefined. В противном случае возвращает значение левого операнда. Этот оператор помогает обрабатывать значения, которые могут быть нулевыми или неопределенными.

```javascript
const a = null;
const b = 'Hello';
console.log(a ?? b); // Output: "Hello"
```

5. **Оператор !!** используется для преобразования значения в логический тип данных. Может быть использован для явного приведения значения к true или false.

```javascript
var value1 = 10;
var value2 = '';
var value3 = 'Hello';

console.log(!!value1); // true
console.log(!!value2); // false
console.log(!!value3); // true
```

- Логическое **НЕ (!):** имеет наивысший приоритет.
- Логическое **И (&&):** имеет более высокий приоритет, чем логическое ИЛИ.
- Логическое **ИЛИ (||):** имеет наименьший приоритет среди логических операторов.
