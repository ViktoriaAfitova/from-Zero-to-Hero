title: call apply bind
---

В JavaScript методы call(), apply() и bind() используются для явной привязки значения 'this' во время вызова функции. Разница в передаче аргументов.

Метод call() - аргументы передаются через запятую.

Метод apply() - аргументы передаются в виде массива.

Метод bind() - возвращает новую функцию. Эта функция может быть вызвана в любом месте приложения.
