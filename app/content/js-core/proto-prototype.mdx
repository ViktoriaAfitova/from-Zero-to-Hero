---
title: proto & prototype
---

**`__proto__`** и **prototype** относятся к механизму наследования в JavaScript.

**prototype** - независимый объект с определённым набором свойств и методов. Есть у класса или функции.

**`__proto__`** - свойство объекта, которое содержит ссылку на его прототип. Оно является частью внутреннего механизма JavaScript и используется для наследования свойств и методов от прототипа.

```JavaScript
const userPrototype = {
  getFullName() {
    return `${this.name} ${this.surname}`;
  };
};

const user1 = {
  name: 'name1',
  surname: 'surname1'
};

const user2 = {
  name: 'name2',
  surname: 'surname2'
};

user1.__proto__ = userPrototype;
user2.__proto__ = userPrototype;

console.log(user1.getFullName()); // name1 surname1
```
