---
title: this
---

**this** используется для обращения к текущему объекту, в контексте которого выполняется код. Значение 'this' зависит от того, каким образом вызывается функция или метод.

Значение "this" может быть определено по следующим правилам:

1. Глобальный контекст:

   - При использовании 'this' вне функции или метода, его значение будет ссылаться на глобальный объект. В браузере глобальным объектом является 'window' (или 'global' в Node.js).

2. Функция:

   - При вызове функции через обычный вызов (например, foo()), значение 'this' будет ссылаться на глобальный объект (в браузере это 'window'). Однако, если функция вызывается в строгом режиме ('use strict'), то значение 'this' будет undefined.

3. Метод объекта:

   - При вызове метода объекта (например, obj.method()), значение 'this' будет ссылаться на сам объект, для которого вызывается метод. В этом случае 'this' используется для доступа к свойствам и методам объекта.

4. Конструктор:

   - При использовании функции как конструктора с оператором 'new' (например, new Foo()), значение 'this' будет ссылаться на новый экземпляр создаваемого объекта. Внутри конструктора 'this' используется для привязки свойств и методов к создаваемому объекту.

5. Привязка значения 'this':
   - В JavaScript существуют методы, такие как call(), apply() и bind(), которые позволяют явно указать значение 'this' для вызова функции. Это полезно, когда необходимо передать определенный контекст внутри функции или метода.
