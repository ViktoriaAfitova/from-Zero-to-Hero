---
title: Comparison Operators
---

- строгое равенство **`===`**: cравнивает два значения на равенство без приведения типов.

- строгое неравенство **`!==`**: сравнивает два значения на неравенство без приведения типов. Возвращает true, если значения или их типы различны, и false в противном случае.

- нестрогое равенство **`==`**: сравнивает два значения на равенство с приведением типов.

```javascript
'2' > 1; // true
'01' == 1; // true '01'-> 1
```

- нестрогое неравенство **`!=`**: сравнивает два значения на неравенство с приведением типов. Если типы операндов различны, они будут приведены к общему типу перед сравнением. Возвращает true, если значения неравны после приведения типов, и false в противном случае.

- больше **`>`**
- меньше **`<`**
- больше или равно **`>=`**
- меньше или равно **`<=`**
