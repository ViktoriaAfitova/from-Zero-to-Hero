---
title: Use Strict
---

**Strict mode** включает более строгую интерпретацию и проверку кода, код выполняется в соответствии с современными стандартами языка JavaScript.

- Заменяет исключениями некоторые ошибки
- Исправляет ошибки, которые мешали движкам JavaScript выполнять оптимизацию ("eval" или "arguments.caller")
- Запрещает использовать некоторые элементы синтаксиса

[Ссылка](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Strict_mode)
