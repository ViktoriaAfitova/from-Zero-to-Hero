**DOM (Document Object Model)** представляет документ в виде дерева объектов и позволяет взаимодействовать с HTML-документом, где каждый элемент HTML-документа является узлом дерева, а атрибуты и текстовые узлы являются его потомками.

С помощью DOM можно изменять содержимое HTML-страницы, добавлять, изменять или удалять элементы, изменять стили и классы, обрабатывать события и так далее. Все это делается с помощью методов и свойств, предоставляемых DOM.

DOM предоставляет различные объекты для работы с элементами HTML-документа. Наиболее распространенные из них:

- **document:** главный объект DOM, представляющий весь HTML-документ. Он предоставляет методы для доступа к элементам на странице и изменения их содержимого.

- **Element:** представляет отдельный элемент HTML. Он предоставляет методы и свойства для работы с атрибутами, классами, стилями и содержимым элемента.

- **NodeList:** представляет коллекцию узлов, которые являются результатом выполнения запросов к DOM. NodeList похож на массив и обычно используется для доступа к группе элементов.

- **Event:** представляет событие, которое происходит на элементе. DOM позволяет прослушивать события и выполнять определенные действия при их наступлении.

DOM предоставляет мощные возможности для манипуляции с HTML-документами. Позволяет создавать динамические и интерактивные веб-приложения, изменять содержимое страницы в реальном времени и взаимодействовать с пользователями.
