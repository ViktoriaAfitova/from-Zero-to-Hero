---
title: Browser events
---

Браузерные события в JavaScript - это события, которые возникают в веб-браузере при взаимодействии пользователя с веб-страницей. Эти события могут быть перехвачены и обработаны с помощью JavaScript, что позволяет создавать интерактивные и отзывчивые веб-приложения.

Вот некоторые из наиболее распространенных браузерных событий JavaScript:

1. События загрузки (Load events):
    - **load:** возникает, когда веб-страница полностью загружена, включая все ресурсы (такие как изображения, стили и скрипты).
    - **DOMContentLoaded:** происходит, когда структура DOM документа полностью построена, без ожидания загрузки внешних ресурсов (например, изображений).

2. События ввода (Input events):
    - **click:** возникает при щелчке мышью на элементе.
    - **input:** происходит при изменении значения элемента ввода (например, текстового поля).
    - **submit:** возникает при отправке формы.

3. События клавиатуры (Keyboard events):
    - **keydown:** возникает при нажатии клавиши на клавиатуре.
    - **keyup:** происходит, когда клавиша на клавиатуре отпущена.

4. События мыши (Mouse events):
    - **mousedown:** возникает при нажатии кнопки мыши на элементе.
    - **mouseup:** происходит, когда кнопка мыши отпускается.
    - **mousemove:** возникает при перемещении указателя мыши над элементом.

5. События фокуса (Focus events):
    - **focus:** возникает, когда элемент получает фокус.
    - **blur:** происходит, когда элемент теряет фокус.

6. События изменения размера окна (Window resize events):
    - **resize:** возникает при изменении размеров окна браузера.
    - **scroll:** происходит при прокрутке содержимого элемента или окна браузера.

#### Способы назначения обработчиков событий:

- Атрибут HTML: onclick="...".
- DOM-свойство: elem.onclick = function.
- Специальные методы: elem.addEventListener(event, handler[, phase]) для добавления, removeEventListener для удаления.
