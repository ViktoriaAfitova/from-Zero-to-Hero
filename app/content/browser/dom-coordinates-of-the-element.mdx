---
title: DOM coordinates of the element on the page
---

#### Координаты элемента на странице можно получить с помощью следующих свойств:

1. **element.getBoundingClientRect():** возвращает объект DOMRect, содержащий информацию о размерах и позиции элемента относительно видимой области окна просмотра.

```javascript
const element = document.getElementById('myElement');
const rect = element.getBoundingClientRect();

console.log(rect.top);       // Верхняя граница элемента относительно видимой области окна
console.log(rect.left);      // Левая граница элемента относительно видимой области окна
console.log(rect.bottom);    // Нижняя граница элемента относительно видимой области окна
console.log(rect.right);     // Правая граница элемента относительно видимой области окна
console.log(rect.width);     // Ширина элемента
console.log(rect.height);    // Высота элемента
```

_Значения, возвращаемые getBoundingClientRect(), указывают на позицию элемента относительно видимой области окна, а не относительно всей страницы. Если страница имеет горизонтальную или вертикальную прокрутку, эти значения будут изменяться при прокрутке страницы._

2. **element.offsetLeft** и **element.offsetTop:** возвращают горизонтальное и вертикальное смещение элемента относительно его ближайшего родительского элемента, который имеет позиционирование отличное от static (по умолчанию).

```javascript
const element = document.getElementById('myElement');
console.log(element.offsetLeft);    // Горизонтальное смещение элемента относительно родительского элемента
console.log(element.offsetTop);     // Вертикальное смещение элемента относительно родительского элемента
```

_offsetLeft и offsetTop возвращают значения только для смещения относительно родительского элемента, а не относительно всей страницы._

_Значения координат могут изменяться при изменении размеров окна браузера или при прокрутке страницы._
