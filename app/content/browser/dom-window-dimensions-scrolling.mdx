#### Cвойства и методы позволяющие получать информацию о размерах окна просмотра и управлять прокруткой окна в DOM.

1. Размеры окна:

- **window.innerWidth:** возвращает ширину окна просмотра браузера, включая ширину вертикальной полосы прокрутки (если она есть).
- **window.innerHeight:** возвращает высоту окна просмотра браузера, включая высоту горизонтальной полосы прокрутки (если она есть).
- **document.documentElement.clientWidth:** возвращает ширину клиентской области документа (видимой части страницы), исключая полосу прокрутки и другие элементы интерфейса браузера.
- **document.documentElement.clientHeight:** возвращает высоту клиентской области документа, исключая полосу прокрутки и другие элементы интерфейса браузера.

2. Прокрутка окна:

- **window.scrollX или window.pageXOffset:** возвращает количество пикселей, на которое окно прокручено по горизонтали.
- **window.scrollY или window.pageYOffset:** возвращает количество пикселей, на которое окно прокручено по вертикали.
- **window.scrollTo(x, y):** прокручивает окно до указанных координат (x, y).
- **window.scrollBy(x, y):** прокручивает окно на указанное количество пикселей относительно текущего положения.

_Размеры и прокрутка окна могут меняться при изменении размера окна браузера или при прокрутке страницы._

3. Метод scrollIntoView():

- **elem.scrollIntoView():** прокручивает содержимое элемента так, чтобы он был видимым в пределах контейнера, в котором он находится. При вызове без аргументов элемент будет выровнен по верхней границе контейнера, если это возможно.
- **elem.scrollIntoView(top):** прокручивает содержимое элемента так, чтобы он был видимым в пределах контейнера, в котором он находится. Если аргумент top равен true, элемент будет выровнен по верхней границе контейнера, а если равен false или не указан, элемент будет выровнен по нижней границе контейнера, если это возможно.

_Метод scrollIntoView() полезен, когда нужно прокрутить страницу к определенному элементу так, чтобы он был полностью видимым внутри контейнера, особенно при работе с длинными списками или таблицами. Параметр top позволяет указать, какую границу элемента выровнять по границе контейнера для обеспечения наилучшей видимости._
