---
title: DOM node properties
---

Узлы DOM - это элементы, которые составляют структуру документа и представляют его содержимое. Каждый узел имеет различные свойства, которые могут быть использованы для доступа к его атрибутам, содержимому и отношениям с другими узлами.

Вот некоторые распространенные свойства узлов DOM:

- **nodeType:** свойство позволяет узнать тип DOM-узла.

  - elem.nodeType == 1 для узлов-элементов,
  - elem.nodeType == 3 для текстовых узлов,
  - elem.nodetype == 8 для комментариев,
  - elem.nodeType == 9 для объектов документа

- **nodeName и tagName:** свойство возвращает название тега (записывается в верхнем регистре, за исключением XML-режима). Для узлов-неэлементов nodeName описывает, что это за узел. Только для чтения.

- **innerHTML:** свойство позволяет получить HTML-содержимое элемента в виде строки и при необходимости изменить.

- **outerHTML:** HTML-содержимое элемента целиком.

- **nodeValue/data:** содержимое текстового узла(текст, комментарий).

- **textContent:** текст внутри элемента.

- **свойство «hidden»** если значение установлено в true, делает то же самое, что и CSS display: none.
